Запуск программ можно осуществить несколькими способами, но
одно должно быть точно - это предустановленный Framework 2.0 и выше,
который можно скачать с сайта  microsoft.com.

Итак, вы можете запускать примеры:
1. с помощью командной строки (компилятора csc.exe)
2. с помощью TextPad (http://www.textpad.com) - условно бесплатно
3. с помощью SharpDevelop - среда разработки с открытым 
   исходным кодом (http://www.icscode.net)
4. с помощью MS Visual Studio 

Рассмотрим первый и последний способы.

---------------------------------------------------------------------
Настройка компилятора командной строки:

1. Проверьте, может уже все настроено. 
   Введите в командную строку:
   CSC /?
   Если высветился список опций настройки - все нормально, если нет,
   следует произвести настройку:

2. В Пуск -> Панель управления -> Система -> Дополнительно -> Переменные среды

3. В окне Системные переменные выберите 'Path' двойным щелчком.
   Там уже имеется множество переменных, разделенных точкой с запятой.

4. Найдите Framework на вашем компьютере и вставьте путь к нему после
   точки с запятой. Например, у меня:

   ;C:\WINDOWS\Microsoft.NET\Framework\v2.0.50727

5. Повторите первый шаг. Если все нормально, вы можете добавить в 'Path' 
   путь к папке с утилитами, чтобы без труда и их можно было запускать с
   командной строки (это необязательно). У меня
   ;C:\Program Files\Microsoft Visual Studio 8\SDK\v2.0\Bin
   
-----------------------------------------------------------------------
Для эффективной работы с програмным кодом могут понадобиться утилиты:

ildasm.exe - исследование содержимого компоновочных блоков. Искать тут:
             (C:\Program Files\Microsoft Visual Studio 8\SDK\v2.0\Bin)
             Если командная строка настроена, просто ввести ildasm.exe
-----------------------------------------------------------------------

Запустим пример "Hello World.css" командной строкой.

1. Создайте рабочую папку (например, D:\Work) и скопируйте туда файл.

2. Откройте командную строку
   Пуск -> Все программы -> Стандартные -> Командная строка

3. Перейдите в папку. Т.е. введите команду 
   > cd D:   - переход на диск D
   > cd \Work - переход в папку Work текущего диска
   > csc HelloWorld.cs
   В этой директории появится файл HelloWorld.exe, который является
   рабочим приложением. Запускаем и смотрим.

Алгоритм запуска программы в Microsoft Visual Studio  2005

1. Открываем VS 2005
2. File -> New -> Project :(C#) Windows -> Empty Project
   Называем проект и выбираем локацию.
3. В откывшемся пректе ищем в левом верхнем углу панель Solution Explorer
   Правой кнопкой по имени проекта -> Add -> New Item -> CodeFile.cs (назовите его)
   Затем скопируйте в него код примера.
4. В главной панели меню:  Debug -> Start Without Debugging.  Все.
